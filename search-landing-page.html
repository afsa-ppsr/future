PPSR search

Search the PPSR to find if there is a security interest registered over particular serial-numbered or other personal property.

What to search on the PPSR

The main search types include motor vehicle, serial number search, organisational grantor and individual grantor search.

PPSR search fees

A fee will apply to searches of the PPSR regardless of the search result.

You can retrieve previous search results and search certificates from the options below. For specific information on relevant concepts such a grantor or serial number, please see the Glossary.

Retrieve search result

This option allows users to retrieve results from previous searches undertaken.

Request search certificate

This search option allows users to request a search certificate for a search result. Search certificates are provided in PDF format.

Reissue search certificate

A search certificate can be reissued on request. You will need the search certificate number, which appears at the top of the Search certificate issued by the PPSR.
