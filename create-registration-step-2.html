Important: Getting the collateral details is key. If you get this information wrong, your registration may not be effective.

You can’t change the collateral details later. If you need to correct them, you will need to create a new registration and pay another fee.

For more information, visit the PPSR website.

Collateral class

If you are uncertain which collateral class to choose, please visit the PPSR website.

The PPS law says that a registration can only record one collateral class so if you need to register against more than one collateral class use the ‘add another registration’ button at the bottom of the page. Additional fees apply for each separate registration.

Secured party group details

You will need to enter your SPG number. You can create an SPG, if you don not have one.

An SPG contains the details of the individuals or organisations with an interest in the property. A secured party is also known as: lender, lessor, consignor, financier, or seller.
